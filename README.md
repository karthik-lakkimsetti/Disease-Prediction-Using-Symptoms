# 📖 Project Overview

Disease Prediction using Symptoms is a Machine Learning application that predicts the most probable disease based on symptoms entered by the user.

The project demonstrates a complete Machine Learning pipeline, including data preprocessing, model training, evaluation, model serialization, and prediction.

---

# 🎯 Objective

- Predict diseases using patient symptoms.
- Build an accurate Machine Learning classification model.
- Demonstrate a complete healthcare prediction workflow.
- Showcase practical applications of AI in healthcare.
